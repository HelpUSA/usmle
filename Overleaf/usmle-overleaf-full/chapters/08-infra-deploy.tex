\chapter{Infrastructure and deployment}
\label{ch:infra}

\section{Vercel deployment}
The project is deployed on Vercel with automatic deployments triggered by GitHub pushes.

\subsection{Deploy trigger policy}
\begin{itemize}
  \item \textbf{Enabled:} GitHub integration (push to \texttt{main})
  \item \textbf{Disabled:} external \emph{Deploy Hooks} (to avoid duplicate deployments)
\end{itemize}
