\chapter{UI/UX blueprint: Session Review}
\label{ch:uiux-review}

This chapter defines the \textbf{Review screen} as the primary learning surface.
The player (pre-submit) is optimized for exam simulation; the review is optimized for learning.

\section{Goals}
\begin{itemize}
  \item Make every answered question a learning opportunity.
  \item Provide explanations for the correct answer and each incorrect option.
  \item Provide study references that can be clicked by the learner.
\end{itemize}
